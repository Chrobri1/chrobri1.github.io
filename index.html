About Me

My name is Christina OBrien. I'm a dublin based digital designer with over ten years experience. I have a passion for creating highly visual and intuitive design solutions for print and web media using a combination of current and emerging technologies.

I have a wide ranging experience from graphic design for print to more recent studies in web technologies including projects in Web Design, Multimedia and Web Application Development. My portfolio is a mix of work created independently and also as part of a studio. I am currently seeking new opertunities in the areas of Graphic Design, Web Deisgn/Development and UXUI Design.

World of Wine

Software Development Project

World of Wine is a e-commerce web application which allows users to make wine purchases online using the integrated paypal system. Other features include Login/Registration and send mail functionality, google authentication, shopping cart and client side administrator only access to the customer and products database. Integration of third party software Review Buddy and inclusion of social media icons to like and share content.
Technologies Used: Visual Studio, 2015, ASP.net, C#, Azure Web Deploy, SQLServer Local DB, HTML5, CSS3, Bootstrap, JavaScript, JQuery, Git Version Control, Adobe Creative Suit. World of Wine
